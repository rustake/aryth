Basic math and statistics tools

## Usage

Add this to your `Cargo.toml`:

```toml
[dependencies]
aryth = ">=0.0.1"
```

and this to your crate root:

```rust
extern crate aryth;
```

## Compatibility

The `aryth` crate is tested for rustc 1.48 and greater.
